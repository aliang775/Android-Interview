#	面试题12 打印1到最大的n位数
---

题目：

>题目：输入数字n，按顺序打印出从1最大的n位十进制数。比如输入3，则打印出1、2、3一直到最大的3位数即999。

我们能写出如下代码：

```
class Solution {
    public int[] printNumbers(int n) {
        if (n <= 0) {
            return null;
        }

      
            int i = 0;
            int yinzi = 9;
            int sum = yinzi;
            while(i < n - 1) {
               yinzi *= 10;
               sum += yinzi ;
               i++;
            }
 
            int[] result = new int[sum];
            int j = 0;
            while (j < sum) {
                result[j] = j+1;
                j++;
            }
            return result;
        }

    }

```
